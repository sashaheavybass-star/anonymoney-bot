# fund-bot

Простой Telegram-бот: счётчик сбора денежных средств с визуализацией в виде «пирога» (donut-диаграммы).

## Установка

```
npm install
copy .env.example .env
```

В `.env` укажите:
- `BOT_TOKEN` — токен бота от @BotFather
- `ADMIN_IDS` — id администраторов через запятую (узнать свой id можно у @userinfobot)

## Запуск

```
npm start
```

## Команды

- `/status` — показать текущий прогресс (картинка-пирог + цифры)

Только для админов (из `ADMIN_IDS`):
- `/setgoal <сумма>` — задать цель сбора
- `/set <сумма>` — задать текущую собранную сумму
- `/settitle <название>` — задать название сбора

## Данные

Хранятся в `data.json` (цель, текущая сумма, название). Файл создаётся при первом запуске автоматически с дефолтными значениями.
